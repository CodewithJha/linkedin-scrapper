ABOUT THE PROJECT

LinkedIn Scraper Project

An intelligent tool for automated job discovery and email finding.

Project Overview

LinkedIn Scraper is a powerful automation tool designed to help job seekers and recruiters efficiently discover job opportunities and find professional contact information. The project combines intelligent web scraping with smart filtering to deliver accurate and relevant results.

Features

  • Job Scraping: Automatically search and filter LinkedIn job postings based on role, location, and posting date
  • Email Finder: Intelligent email pattern matching to discover professional work emails
  • Smart Filtering: Advanced filtering options to find the most relevant opportunities
  • CSV Export: Export all scraped data in CSV format for easy analysis
  • Automated Scheduling: Set up automated scraping sessions with customizable intervals
  • Deduplication: Smart deduplication to avoid processing the same jobs multiple times

Technology Stack

Built with modern web technologies including Node.js, Playwright for browser automation, and Express.js for the API server. The frontend uses vanilla JavaScript with a modern, responsive design that works seamlessly across all devices.

Usage

Simply enter your search criteria (job role, location, etc.) and click "Start Scraping". The tool will automatically browse LinkedIn, collect job postings, and export them to CSV. For email finding, provide a person's name and company domain to discover their likely work email.

Open Source

This project is open source and available on GitHub. Contributions, issues, and feature requests are welcome. The project is maintained to help the community with job searching and professional networking automation.
